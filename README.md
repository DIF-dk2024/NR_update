# Рынок Квартир Астаны — 3 кнопки → 3 страницы (редактирует админ)

## Что это
Публичная главная показывает 3 кнопки:
- Подписаться в Telegram
- Эксклюзивная Аналитика
- Купить Курс

Каждая кнопка открывает отдельную страницу (карточку), где есть:
- Заголовок
- Описание
- Поле ссылки (URL)
- Несколько вложений (файлы/видео)

Редактирование доступно только администратору (секретный вход).
