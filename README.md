# Рынок Квартир Астаны — 3 кнопки → 3 страницы (редактирует админ)

## Что это
Публичная главная показывает 3 кнопки:
- Подписаться в Telegram
- Эксклюзивная Аналитика
- Купить Курс

Каждая кнопка открывает отдельную страницу (карточку), где есть:
- Заголовок
- Описание
- Поле ссылки (URL)
- Несколько вложений (файлы/видео)

Редактирование доступно только администратору (секретный вход).

## Важные URL
- Главная: `/`
- Страницы: `/p/telegram`, `/p/analytics`, `/p/course`
- Секретный админ-вход: `/karna1203-admin-login`
- Админ-панель страниц: `/admin/pages`

## Render env vars
Обязательно:
- `SECRET_KEY`
- `ADMIN_PASSWORD`
- `DATA_DIR=/var/data`
- `UPLOADS_DIR=/var/data/uploads`

Опционально:
- `MAX_CONTENT_LENGTH` (лимит загрузки в байтах, например 31457280 для ~30MB)
